    The Blog Project

    The Blog Project is my personal blog where I write about coding and technology. You may ask yourself, "Why create your own blog software?" The answer is: Being able to code gives you the opportunity to construct the software you want to use. I built my own blog because I could, and I wanted to blog about my passion: Coding. The Blog Project is built with ASP.NET MVC, C# and SQL. A role-based security system was implemeted to create permission-based roles. The roles of Administrator and Moderator restrict what the user can see and do. The software allows the Administrator to create Blog articles, and Moderators can moderate or delete comments.

    The front end was developed with ASP.NET MVC, HTML, CSS, Bootstrap and JavaScript. The back end utilizes ASP.NET MVC and C#. The database was built with SQL Server and is hosted on Heroku. Please visit the site, read an article and leave a comment!
